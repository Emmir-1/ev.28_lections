
Linux commands
Ctrl + alt + T - команда открытия терминала 

~ - символ домащней директории
/ - символ корневой директории  

. - обозначает текущую директорию
.. - обозначает предыдущую директорию 

1. Ls - команда для вывода всех директории(папка) и файлов в текущей(рабочей) директории в терминале 

2. cd (change directory) - комната для перехода из одной директории в другую 

3. mkdir - создает директории 

4. touch - создает файл 

5. rmdir - удаляет пустую директорию

6.rm - удаляет файл 

7. rm -rf - удаляет вообще все(принудительное удаление)

8. ls -a - показывает скрытые и обычные файлы и директории 
   ls -la - показывает подробную информацию 

9. pwd (print working directory) - выводит путь до текущей директории 

10.cp - копирование файл или директорию в указаное место 
   cp <название> <куда>

11.mv - команда для перемещение файла или директории, а также для переименовывание файла или директории
        Переименовывание: mv john.txt jamie.txt
        Перемещение: mv john.txt directory/

12. cat - показывает содержание файла 

13. nano - открывает файл для редактирование в графическом интерфейсе nano
*если нет файла, то nano создаст новый 

14. echo - команда для быстрой записи текста в файл 
    echo текст > файл 

15. clear - очищение терминала
